# poker

    It's a poker game website.

[Visit website](https://game.dynedyne.com)

## Stack

- React (FE)
- Nodejs (BE)

## Get started

### `frontend`

```bash
git clone https://github.com/DynePark9111/poker.git
yarn add
touch .env
yarn dev

```

### `backend`

[Github: poker_be ](https://github.com/DynePark9111/poker_be)

### `.env`

```
VITE_URL = http://localhost:8080
```
